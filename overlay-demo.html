üöÄ Revolutionary Approach

The Overlay Strategy solves the fundamental problem of skeleton loading: getting the exact dimensions right.

Instead of guessing component sizes or maintaining complex mapping tables, we:

  1. Render the actual component invisibly to establish perfect dimensions
  2. Overlay a skeleton that matches exactly
  3. Get pixel-perfect results automatically

üéÆ Usage Examples

Basic Usage:

import { Skeletize } from '@sedonawebservices/skeletize-chakra-ui';

// Pixel-perfect skeletons for any component
<Skeletize loading={true} strategy="overlay">
  <Button size="xl" px={8}>Large Button</Button>
  <Heading size="lg">Perfect Heading</Heading>
  <CustomComponent width="300px">Even custom components!</CustomComponent>
</Skeletize>

Complex Layouts:
